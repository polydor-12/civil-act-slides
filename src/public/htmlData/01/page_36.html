지상권자의 갱신청구권

  • 의의 : 존속기간이 만료한 후 지상물이 존재하는 때에는 지상권설정자에게 계약갱신 청구가능
  • 법적 성격 : 청구권
  • 요건 : 지상권 존속기간 만료 + 지상물의 현존
  • 효과 : 지상권설정자는 갱신청구에 응해야 할 의무가 있음 (단, 정당한 이유가 있으면 거절 가능)

지상권자의 지상물매수청구권

  • 의의 : 지상권설정자가 갱신청구를 거절한 경우 지상권자는 지상물매수청구권을 행사할 수 있음
  • 법적 성격 : 형성권
  • 요건 : 갱신청구의 거절 (지상권설정자가 갱신을 거절한 경우)
  • 효과 : 청구에 의해 지상권설정자는 지상물을 매수할 의무가 발생함 (적정한 가격으로 매수)
